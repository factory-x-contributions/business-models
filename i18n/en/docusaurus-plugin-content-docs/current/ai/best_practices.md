## Context Window & Document Backup

Every AI chatbot has a **limited context window**. After a certain number of messages, the capacity may be exhausted.

<div style={{background: 'var(--ifm-color-emphasis-100)', borderRadius: '8px', padding: '1rem 1.25rem', marginTop: '1rem'}}>

**What you should keep in mind:**

- **After each room:** Save results as an output document
- **Check capacity:** Open a new chat if needed
- **Upload outputs:** Provide saved documents in the new chat

</div>

## Recommended LLM Models (as of February 2026)

| Model | Strengths |
|--------|---------|
| [**GPT-5**](https://chat.openai.com/) | Comprehensive analyses, management-ready outputs |
| [**Claude Sonnet 4.6**](https://claude.ai/) | Precise answers, fast, ideal for professional work |
| [**Claude Opus 4.6**](https://claude.ai/) | Highest quality, complex tasks, Agentic AI |
| [**LLaMA 4**](https://www.meta.ai/) | Open-source, company-internal deployments |
| [**Mistral Large**](https://chat.mistral.ai/) | European alternative, multilingual, GDPR-compliant |

:::tip Model Settings
Use the **"Auto" mode**, not exclusively the "Thinking" mode. Use neutral chat settings without a specific system prompt.
:::

:::warning Data Privacy
Avoid using public chatbots with sensitive company data. Prefer company-internal LLMs or secure platforms that handle data confidentially.
:::
